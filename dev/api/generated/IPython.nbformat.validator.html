Module: nbformat.validator¶

3 Functions¶

IPython.nbformat.validator.isvalid(nbjson)¶

Checks whether the given notebook JSON conforms to the current notebook format schema. Returns True if the JSON is valid, and False otherwise.

To see the individual errors that were encountered, please use the validate function instead.

IPython.nbformat.validator.validate(nbjson)¶

Checks whether the given notebook JSON conforms to the current notebook format schema, and returns the list of errors.

IPython.nbformat.validator.resolve_ref(json, schema=None)¶

Resolve internal references within the given JSON. This essentially means that dictionaries of this form:

{“$ref”: “/somepointer”}

will be replaced with the resolved reference to /somepointer. This only supports local reference to the same JSON file.
